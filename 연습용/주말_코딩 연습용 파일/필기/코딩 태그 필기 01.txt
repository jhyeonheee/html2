<!DOCTYPE html> : HTML5 문서 선언
	<html> : 언어 속성 사용 -> 언어값 설정
		<head> : 웹페이지의 정보 정의
			<title> : 웹 브라우저 제목!
			<meta> : 문서 제작자, 검색 키워드, 웹 페이지 정보, 리프레시 작성
			<base> : URL 지정
			<style> : 문서 장식 프로그램 위치하는 공간(ex.CSS3) / html 문서 내 작성 시 사용
			<link> : 외부 문서,사이트 지원 연결 / 외부 스타일 파일 제작 시 사용
			<script> : 다양한 프로그램 위치하는 공간(ex.자바스크립트) / 내부,외부 분리 가능
			<body> : 웹 화면 노출 정보 작성하는 공간(문서 본문 작성)

<p> : 문단구분
<br> : 줄 바꿈 (단독태그)
            <br />로 표시 권장  
&nbsp; : 공백
<!-- ~ -- >> : 주석 / <style> 안에서는 *~*
<h1>~<h6> : 문서의 단계별 제목 설정
                        중복 사용 불가(1회만 가능)
                        숫자와 제목 크기는 반비례(1이 제일 큼)
 text-align: center; : 텍스트 가운데 정렬(가로)
line-height: (높이)300px; : 텍스트 가운데 정렬(세로)

목록태그
	1. 무순서 목록  <ul>~<li> : 불릿 생성 (목록태그)
		     	        <ul><li></li></ul> 처럼 조합 필수
 	                     	        순서없는 목록태그 

	2. 순서 목록     <ol>~<li> : 순서있는 목록태그(1,2,3 ...)
		   	        <ol><li></li></ol> 조합

	3. 정의형 목록  <dl><dt><dd></dl> : 제목과 그 정의 설명
		 		         <dl>
				   	         <dt>제목</dt>      
				   	         <dd>제목의 정의</dd>
			  	         </dl>


그룹태그
	<div></div> : 요소를 감싸줄 때 사용(그룹)

블록태그 : 좀 더 큰 태그 / 부모가 가진 전체 넓이 차지(ex. h1~h6태그, div태그)
인라인 태그 : 작은 태그 / 본인들만 적용(ex. 백그라운드 컬러,em태그)

폰트 스타일 태그
<em> : 이탤릭체로 표기

07.08 - 테두리 속성(p.237)
<border-width> :테두리 두께 설정
<border-color> :테두리 색상 설정
	border-color: #000000;
	border-color: rgb(255, 255, 0);
	border-color: rgba(255, 255, 0); ->(투명도 적용)
<border-style> : 테두리 스타일 적용
	border-style: none; -> 테두리 제거
		      hidden -> 테두리 감추기
		      double -> 테두리 이중선 지정
		      groove -> 테두리 오목한 선으로 지정(홈이 파인 듯 입체적)
		      dotted -> 테두리 점선 지정
		      ridge -> 테두리 볼록한 선으로 지정(튀어나온 듯 입체적)
		      dashed -> 테두리를 파선(긴 점선)으로 지정
		      solid -> 테두리를 실선으로 지정
		      inset -> 테두리 안쪽이 오목한 선으로 지정
		      outset -> 테두리 안쪽이 볼록한 선으로 지정
<inherit> : 부모값 없애기
	color: inherit; -> 컬러값 초기화
<border-(top/bottom/left/right> : 테두리 한쪽 적용

07/18 필기(텍스트 배경 이미지 적용)
1. 이미지 적용 > background-image: url('img/강가.jpg');

2. 적용한 이미지 사이즈 설정 >  background-size: 200px; 
	  /* cover(최다사용,비율에 맞춰서 가득 채워짐, 이미지 일부 짤림 가능), contain(가로세로 비율 유지하면서 가득채움) */

3. 이미지 반복 설정 >  background-repeat:repeat; 
	 /*repeat(기본값),no repeat(없애기),방향설정> repeat-x(x축 기준 이미지 반복), repeat-y(y축 기준 이미지 반복) */

4. 이미지 위치 변경 >  background-position:20px 100px ;/* 첫번째 값 가로, 두번째 세로값 + %로 설정 가능(50%= 중앙정렬)+명령어로 정렬 가능(center center = 중앙)*/
	x축 > left, center, right  /  y축> top, center, bottom    

5. 효과 한번에 적용하는 방법 >  background: url('img/해변\ 파도.jpg') center center / cover no-repeat;
		(선 위치 / 후 사이즈 순서 바뀌면 ㄴㄴ) */  /* 배경:이미지 경오 x축 위치 y축 위치 / 사이즈 이미지 반복 */       

6. 배경 이미지 고정 > background-attachment: fixed;
	/* 배경 이미지 고정시킴+ 속성값만 background:;안에 넣어 설정 가능(마지막 순서) */

7. 그라데이션 배경 삽입 > '그라디언트 IO' 사이트 가서 코드 복붙

8. 블러효과 .  backdrop-filter: blur(5px);
